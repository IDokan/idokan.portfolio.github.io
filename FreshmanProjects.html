Freshman Projects

In DigiPen's Freshman courses, I should do a half year project twice, Minotaur and PhonePong. PhonePong is made by given framework.

In gam100 class, my team made a command prompt game which is Minotaur.

I was a tech-manager in "Due Date" which is the team name. The team was a four person team.

At the left top green box is a maze which player must escape from. The circle in the center is a player and filled rectangle is a monster who chase the player.

I implemented monter's AI using A* path finding algorithm. Furthermore, I made my list structure because using STL is not available in the project.

In gam150 class, I and teammates made "PhonePong" with given framework engine.


In this project, I became a producer to learn team and resource management because I had thought the last project could be better if I could deal with problems about team management properly.

Even though I was a producer, I implemented platform generator, background update logic, fever mode, and VFX when player's HP is running out. Furthermore, I have learned how to manage team problems and time scheduling using Trello as I wanted in this project.

All tasks are located on 'Finished' section because this project is completely done.
